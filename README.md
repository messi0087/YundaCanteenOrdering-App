# YundaCanteenOrdering-App


这是机场订餐的app需要配合另外一个后端项目进行使用，可以直接安装本项目中的apk



还可以下载和使用adnroid studio进行虚拟机的使用


首先git或下载后

yarn install 或npm install

启动后端项目后

在config/default中配置好port和本机的ip

连接后及可以使用项目/**-----****   别忘了后端配有数据库，先执行sql配置数据库我使用的是mysql的数据库
